# Maze Generator and Solver

This project generates and solves rectangular mazes using stack-based depth-first search (DFS) and backtracking algorithms.

## Features

- Random maze generation
- Dynamic wall removal animation
- Maze solving visualization
- Dead-end backtracking visualization
- OpenGL rendering using PyOpenGL

## Algorithms Used

### Maze Generation
Recursive backtracking DFS with stack support.

### Maze Solving
Backtracking depth-first search.
